# PalmaSat v3.0 · Finca El Porvenir

Dashboard satelital para palma africana con datos en tiempo real desde Google Earth Engine.

## Stack

- **Frontend**: Next.js 14 + React 18
- **API**: Google Apps Script
- **Datos**: Google Sheets
- **Satelital**: Sentinel-2 via GEE

## Deploy en Vercel

1. Conecta este repo a Vercel
2. Deploy automático
